Importar dados do iClinic

Como exportar do iClinic

  1. Clique na foto de perfil no canto superior direito → Configurações da conta
  2. No menu lateral esquerdo, clique em Gestão de dados → Exportar dados
  3. Clique em Exportar dados e configure: profissional de saúde, período, formato CSV, dados de exportação: Pacientes e Agendamentos
  4. Selecione os 2 arquivos gerados abaixo
  5. Clique em Importar / Atualizar — registros já existentes serão atualizados, sem duplicatas

Pode rodar quantas vezes quiser. Dados corrigidos manualmente nos pacientes são preservados automaticamente.

Zona de perigo — Apagar todos os dados

Remove permanentemente todos os registros de recebimentos, despesas, notas, pacientes e agendamentos. Use apenas para reiniciar do zero antes de uma nova importação limpa.

Importação única — Planilha de Pagamentos

Importa dados históricos de pagamentos (set/2025 em diante) a partir do CSV da secretária. Esta operação é pontual — não repetir após a importação inicial.

  1. Selecione o arquivo CSV com os dados de pagamento
  2. Revise os registros e vincule os pacientes não identificados automaticamente
  3. Clique em Confirmar Importação para gravar no sistema
